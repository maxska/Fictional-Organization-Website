Java

Java är ett språk som började utvecklas under 1990-talet och gavs ut under namnet Java år 1995.

Java är ett av de mest populära programmeringsspråken just nu, och har varit det länge. Detta framförallt tack vare dess virtuella maskin, JVM (Java Virtual Machine) som sköter all minneshantering. Med hjälp av JVM kan Java-program köras på alla operativsystem som kan köra JVM. Java är populärt även tack vare att det är specifikt objektorienterat, vilket är ett mycket populärt programspråkparadigm.
