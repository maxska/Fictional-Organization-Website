JavaScript

JavaScript (JS) är ett språk som gavs ut 1995 i syfte att öka funktionalitet på webbsidor.

JS körs oftast direkt med en motor inbyggd i webbläsaren. En stor fördel med JS är att man på ett smidigt sätt kan kontrollera utseendet i HTML-element. Dessutom kan man som programmerare skapa grafiska användargränssnitt med enbart HTML, CSS och JS.
